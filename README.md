# Attendance System

## 📌 Description

This project is a smart attendance system designed to track employee/student attendance using modern technologies like GPS and Face ID.

## 🚀 Features

* User Login / Logout
* Record Attendance (Check-in / Check-out)
* View Attendance Records
* Generate Reports
* Admin Management

## 🛠️ Technologies Used

* Programming Language: (Java / Python / C++)
* UML Diagrams: draw.io / StarUML
* Version Control: Git & GitHub
